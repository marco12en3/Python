# dataviz

Module for urban journey to visualize data from channels in graphs

example for ujml file:

    <?xml version="1.0"?>
    <ujml version="0.0.1">
        <gui inp="foo">
            <graphwindow title="sensors" checked=true>
                <graph tittle="accel" checked=true>
                    <curve inp="foo"/>
                    <curve inp="foo"/>
                    <curve inp="foo"/>
                </graph>
            <graphwindow title="states" checked=false>
        </gui>
    </ujml>
